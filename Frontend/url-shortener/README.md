# URL Shortener

This project is a simple URL shortener built with React, featuring a dark theme with violet accents. Users can input a long URL, optionally specify a custom short URL, and view the shortened URL.

## Features

- Input for long URLs
- Optional custom short URL input
- Display of the shortened URL
- Dark theme with violet accents

## Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/url-shortener.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd url-shortener
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the application**
   ```bash
   npm start
   ```

The application will be available at `http://localhost:3000`.
